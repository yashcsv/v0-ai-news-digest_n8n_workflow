Built an AI agent in n8n that pulls live news based on user topics, filters noise, and turns it into clean, readable email digests automatically.

Wired together NewsAPI, an LLM (Gemini/OpenAI), and email delivery into a single flow, cutting manual news tracking down to near zero.

Added ranking logic (freshness, relevance, source quality) so the most important stories consistently surface at the top.

Used memory to keep responses consistent across runs and avoid repetitive or low-quality summaries.
Designed the workflow to be easily extendable—new data sources or tools can be plugged in without rewriting the pipeline.
